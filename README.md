# 简洁待办清单 - Todo App

一个现代化、美观的待办清单网页应用，使用原生 JavaScript 和 Tailwind CSS 构建。

## 功能特点

- ✅ **添加任务** - 输入框 + 添加按钮，支持按 Enter 键快速添加
- ✅ **标记完成** - 点击任务文本或复选框标记为完成/未完成
- ✅ **删除任务** - 每个任务都有删除按钮
- ✅ **任务筛选** - 查看全部、进行中或已完成的任务
- ✅ **数据持久化** - 使用 localStorage 保存任务，刷新页面不丢失
- ✅ **任务统计** - 显示总任务数和已完成任务数
- ✅ **批量操作** - 清空已完成任务或清空所有任务
- ✅ **响应式设计** - 适配桌面端和移动端
- ✅ **美观界面** - 现代化设计，平滑过渡动画

## 技术栈

- **HTML5** - 语义化标记
- **Tailwind CSS** - 现代化 CSS 框架
- **原生 JavaScript** - 无框架依赖
- **Font Awesome** - 图标库
- **Google Fonts** - Inter 字体

## 使用方法

### 在线使用
直接打开 `index.html` 文件在浏览器中运行。

### 本地开发
无需构建步骤，直接在浏览器中打开即可。

### 功能说明

1. **添加任务**
   - 在输入框中输入任务内容
   - 点击"添加"按钮或按 Enter 键

2. **标记完成**
   - 点击任务文本或前面的复选框
   - 已完成的任务会显示删除线

3. **删除任务**
   - 点击任务右侧的 × 按钮
   - 支持清空已完成任务或所有任务

4. **任务筛选**
   - 使用顶部的筛选按钮查看不同类型任务
   - 全部 / 进行中 / 已完成

5. **数据保存**
   - 所有操作自动保存到浏览器本地存储
   - 刷新页面或关闭浏览器后数据不会丢失

## 设计特点

- **现代化界面** - 使用卡片式设计、阴影和圆角
- **响应式布局** - 适配各种屏幕尺寸
- **平滑动画** - 添加、删除、状态切换都有动画效果
- **直观反馈** - 操作后显示通知提示
- **无障碍设计** - 支持键盘操作和屏幕阅读器
